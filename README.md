# warm emptiness

A minimal Spicetify theme — deep blacks with soft sakura pink accents.

![Theme preview](preview.png)

## Install

1. Install [Spicetify](https://spicetify.app/docs/getting-started)
2. Clone or download this repo into your Spicetify themes folder:

   ```bash
   # Windows
   cp -r warm-emptiness "$env:APPDATA/spicetify/Themes/"

   # Linux/macOS
   cp -r warm-emptiness ~/.config/spicetify/Themes/
   ```

3. Apply the theme:

   ```bash
   spicetify config current_theme warm-emptiness
   spicetify apply
   ```

## Features

- Deep black backgrounds for high contrast
- Sakura pink accent palette
- Frosted glass effects on top bar and overlays
- Subtle noise texture for depth
- Smooth transitions and hover animations
- Outfit font family
- Full Encore design system overrides (no Spotify green leaking through)
- Custom scrollbars, context menus, tooltips, and modals
